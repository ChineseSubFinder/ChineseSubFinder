# 电影和连续剧目录结构示例

如果你使用的是 sonarr 和 radarr 去下载电影和连续剧的话，应该就是类似以下的目录结构，也是本程序推荐的结构。

> 如果不是这样的目录结构，遇到问题可能是很正常的。原由很简单，现在市面上的多款字幕搜索工具因为要考虑的情况太多，导致他们的搜索结果不太稳定，源头上本程序希望能够统一的规范下去完成这个搜索的任务。而且整理好的数据以后用起来就舒服嘛···

## 电影

如果你现在的电影都是零散的丢在相同的一级目录下的，个人还是建议使用 [TinyMediaManager](https://www.tinymediamanager.org/)，进行削刮一次。合理的存储结构对你后续的存储和后续其他程序的对接都是优势。如果未能按下面示例的结构去存放，那么这些视频很可能就会被跳过的（优先是解析视频根目录的几个配置文件，读取 IMDB 信息，其次才是使用文件名去搜索）。

> 削刮器记得选择把相应的削刮信息保存到视频文件夹目录下。会出现类似以下几个文件：
>
> * movie.xml
> * The Night House (2021) Bluray-1080p.nfo
>
> 这些文件对后续的搜索很重要，虽然就算没有也能通过视频名称去搜索，但是效果一般不佳。

以下是示例：

```
Video/电影/
	├─The Night House (2021)
	│      clearlogo.png
	│      fanart.jpg
	│      landscape.jpg
	│      movie.xml
	│      poster.jpg
	│      The Night House (2021) Bluray-1080p.chinese(简英,subhd).ass
	│      The Night House (2021) Bluray-1080p.chinese(简英,zimuku).default.ass
	│      The Night House (2021) Bluray-1080p.mkv
	│      The Night House (2021) Bluray-1080p.nfo
	│      
	└─Dune (2021)
			clearlogo.png
			Dune (2021) WEBRip-1080p.chinese(简,shooter).srt
			Dune (2021) WEBRip-1080p.chinese(简,xunlei).srt
			Dune (2021) WEBRip-1080p.chinese(简英,shooter).ass
			Dune (2021) WEBRip-1080p.chinese(简英,xunlei).ass
			Dune (2021) WEBRip-1080p.chinese(简英,zimuku).default.ass
			Dune (2021) WEBRip-1080p.mkv
			Dune (2021) WEBRip-1080p.nfo
			fanart.jpg
			landscape.jpg
			movie.xml
			poster.jpg
```



## 连续剧

这里是有文档描述的，[连续剧目录结构要求](https://github.com/allanpk716/ChineseSubFinder/blob/docs/DesignFile/%E8%BF%9E%E7%BB%AD%E5%89%A7%E7%9B%AE%E5%BD%95%E7%BB%93%E6%9E%84%E8%A6%81%E6%B1%82.md)。这个是必须遵守的存储结构，有这个结构就默认你经过了削刮器处理，否则也是不能正确去搜索字幕的。

> 如果你不是使用 sonarr 去下载连续剧的，个人还是建议使用 [TinyMediaManager](https://www.tinymediamanager.org/)，削刮一次。一般来说就符合以下的结构了。注意保存削刮的信息到当前的连续剧目录。应该会出现：
>
> * tvshow.nfo 
> * 爱，死亡和机器人 - S01E01 - 索妮的優勢.nfo
> * season.nfo 
> 
> 类似的信息出现在相应的文件夹中。这个很重要！
